Low Cost Telemontoring System for Mines

2015-16 (Masters Project)

This was Masters thesis project; in the value chain of IoT, this project was focused mostly on the development of sensors frontend, a low-cost localised networking solution of those sensors. The solution proposed in this work addressed issues like easy serviceability of this system by keeping development environment of the proposed prototype the Arduino IDE. As Arduino programming environment is much easy to learn and expertise, this work claimed that the proposed openware technology in the thesis work could diffuse much faster in the industrial sectors of Indian and other developing country much more rapidly than its currently existing counterparts.

SAR Interferometry

2014 (B.Tech Project)

“Study on Applicability of Remote Sensing in Detection of the Ground Deformation”: this project started with reviewing several case studies, and technical articles were reviewed, to cite a comparison between conventional methods of ground deformation monitoring, and with ‘Synthetic Aperture Radar Interferometry’ (InSAR). This project dealt with the basics of the RADAR operation, data storage and file handling of radar data. Data processing and data interpretation algorithms for radar imaging. Finally, an attempt was made to do a simulation for ‘Side Looking Aperture Radar’ (SLAR) which shall be capable of demonstrating both Real Aperture Radar (RAR) and Synthetic Aperture Radar (SAR) data accusation procedures. However, at that point of time only first prototype of the program was presented that compromised many desired properties in programs generated ‘Single Look Component’ (SLC) Images.

Hyper-Spectral Satellite Remote Sensing for Mineral Identification

2013 (3rd Yr. Summer Project)

The primary aim of this project was to code programs for a computational tool which shall be capable of demonstrating data processing for Hyper-Spectral image data. Whereas in the case of software, it just does the processing, that is often like a black-box to an operator. Thus, the purpose of this project was to build a framework that gives operator flexibility of manually see and tweak the data at each step of the process; and also enables the user to insert some of his/her designed algorithms in between, for the sake of experimentation and research. Implementations of these analysis were comprised of technique like PCA, n-D Clustering, VCA including other multivariate statistical analysis, data visualisation techniques, data interpretation etcetera.
